# Parisa Darab — Developer Portfolio

My personal portfolio, built from scratch with the React ecosystem. A fast,
responsive, accessible single-page site showcasing my work as a full-stack
JavaScript developer.

🔗 **Live site:** https://parisadarabdev.vercel.app/

## Built with

- **Next.js** (App Router) — server components for fast loads and good SEO
- **TypeScript** — type-safe components throughout
- **Tailwind CSS** — utility-first styling with a consistent design system
- **next/image** — optimised images

## Features

- Fully responsive, mobile-first layout
- Light & dark mode support
- Accessible: semantic HTML, keyboard navigation, visible focus states
- Smooth-scrolling navigation with a sticky header
- Reusable, typed components (Navbar, Hero, About, Experience)

## Run locally

```bash
# clone the repo
git clone https://github.com/ParisaDarab/portfolio.git
cd portfolio

# install dependencies
npm install

# start the dev server
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).
